MusicLessonPlanner: Help and Support

Create a new account
  1. Click here to create a new account.
  2. You will then be prompted to enter your name and the e-mail you would like to be associated with the account, followed by a password for your account.

    3. Example input:

    • Mary Jane
    • mj@email.com
    • Password123
  3. Once you have entered all of this information correctly, select Register. If any of your input is not the correct format an error will pop up to notify you. When set up is complete, you will be brought to your new dashboard where you can proceed with adding students and checking your schedule for the day.
Logging in to your account
  1. Click here to log in to your account.
  2. You will then be prompted to enter your user name and password. Your user name is the email address associated with the account.

These instructions assume that the user has already created an MLP account. If you are unsure of how to do this or need additional help, click the Account Creation tab above.

All of these instructions begin from the main dashboard view of your account. To get to this view, select Main page or the music note symbol from the tab of available pages along the left side of your screen. If this tab is not visible, you can expand it using the small green square at the top left of the window.

Add a new student record
  1. Click the New student button. If this button is not immediately visible, you may need to scroll down past the Today section and to the bottom of the Students section.
  2. This will open a page where you can enter the new student's information.
  3. Once you have entered all of this information correctly, select Save to create the new student account. If any of your input is not the correct format an error will pop up to notify you. If you decide to cancel the new student account creation, select Cancel and the page will exit without storing any information.
  4. After saving the new student record, you will be brought to your updated dashboard. Here you can see the name and instrument of your new student in your list of students. If they have a lesson scheduled for today, you will also see their name and time under the Today section of your dashboard.
 View a student record
  1. Locate the student account you wish to view.
    If the student has a lesson today, their name should be visible in the Today section of your main dashboard view.
    If the student does not have a lesson today, their name will be in the Students section of your main dashboard. If this section is not immediately visible, you may need to scroll down past Today.
  2. Once you have located the student account you wish to view, select anywhere in the row of the student you wish to view.
  3. This will direct you to the Student Records page where you can see more detailed information about the student account which you selected.
  4. To change between student views, you can select a different student from the list of students. For users on large screens this list will be visible to the right of the current student account. For users on smaller screens this list is located beneath the current student account.
 Edit a student record
  1. Navigate to the detailed view of the student account you wish to view by selecting anywhere in the row containing their name.
  2. From here, select the Edit student button.
  3. This will open a page where you can modify the student's information or update their lesson notes.
  4. Once you have entered all of this information correctly, select Save Changes to record your edits. If any of your input is not the correct format an error will appear to notify you. If you decide to cancel your edits, select Cancel and you will be returned to the dashboard.
  5. After editing the student record, you will be brought to your dashboard.
 Delete a student record
  1. Student accounts can be deleted from the main page, or from the detailed student record view. Deleting from either of these views requires the same steps.
  2. Locate the student who's account you wish to delete. If the list of current students is not visible you may need to scroll down to see it.
  3. Select the red Delete button on the right side of the row correlating to the student who's account you wish to delete. This will permanently delete that student's account.
 Reschedule a lesson
  1. Lessons can be rescheduled by selecting a lesson from the detailed student view or Today's lessons on the main dashboard.
  2. After you have selected the lesson which you wish to reschedule and opened the corresponding page, select the Reschedule Lesson button. A page will open prompting you to select a new date and time for the rescheduled lesson.
  3. Once you have picked the new date and time for this lesson, select Save Changes. Otherwise, if you do not wish to reschedule the lesson, select Cancel.
 Create lesson notes
  1. Notes for each individual lesson can be updated by selecting the lesson from the list of lessons in the detailed student view or from today's lessons if it appears there.
  2. Select the Edit Lesson Notes button to modify the notes for this lesson.
  3. You will be taken to a page where you can enter the notes and select Save changes to save these notes or Cancel to be returned to the lesson view without saving the notes.
